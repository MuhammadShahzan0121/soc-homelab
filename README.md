# SOC Home Lab

## Overview

This project is a beginner SOC (Security Operations Center) home lab built using UbuntuARM, Windows, Elasticsearch,
Kibana and Winlogbeat.

The goal of this lab is to:

* Learn SIEM administration
* Monitor Windows security logs
* Detect suspicious activity
* Practice incident investigation
* Improve cybersecurity documentation skills

## Technologies Used

* UbuntuARM64 Server
* Windows 10/11
* Elasticsearch
* Kibana
* Winlogbeat
* PowerShell
* Windows Event Logs
