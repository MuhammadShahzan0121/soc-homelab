# SOC Home Lab

## Overview

This project is a beginner SOC (Security Operations Center) home lab built using Ubuntu, Windows, and Wazuh SIEM.

The goal of this lab is to:

* Learn SIEM administration
* Monitor Windows security logs
* Detect suspicious activity
* Practice incident investigation
* Improve cybersecurity documentation skills

## Technologies Used

* Ubuntu Server
* Windows 10/11
* Wazuh
* PowerShell
* Windows Event Logs
