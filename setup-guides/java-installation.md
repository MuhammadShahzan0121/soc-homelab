# Java Installation

## Objective

Install Java dependencies required for Elasticsearch.

---

## Installation Command

```bash id="6m4q1x"
sudo apt install openjdk-17-jdk -y
```

---

## Java Verification

Command Used:

```bash id="1q8m5x"
java -version
```

---

## Result

Java installed successfully and the version information was displayed correctly.
